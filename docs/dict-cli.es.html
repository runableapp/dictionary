dict-cli Guía completa

CLI de diccionario SQLite offline con modos REST API (rest) y MCP stdio (mcp).

  • Búsqueda offline
  • Modo REST API
  • Integración MCP

D I C T - C L I   M A N U A L

1) Funciones

  • Búsqueda por archivo/directorio SQLite
  • Flujo de búsqueda: predeterminado --term y búsqueda semántica --query
  • Modos: CLI / REST API / MCP

2) Requisitos

  • Go 1.25+
  • Ruta de DB de diccionario
  • /path/to/dictionary_fts5.db o /path/to/dicts

3) Ejemplos CLI

dictcli --db /absolute/path/to/dictionary-data/dictionary_fts5.db groups
dictcli --db /absolute/path/to/dictionary-data/dicts --term 사전
dictcli --db /absolute/path/to/dictionary-data/dicts --term 'dict*'
dictcli --db /absolute/path/to/dictionary-data/dicts --query dictionary

Comodines: * (cualquier secuencia), ? (un solo carácter).

4) Configuración

  1. --db
  2. DICTCLI_DB_PATH
  3. config.ini

5) REST API

dictcli --db /absolute/path/to/dictionary-data/dicts --limit 20 --timeout 60 rest --host 127.0.0.1 --port 8080

6) MCP

dictcli --db /absolute/path/to/dictionary-data/dicts --limit 20 --timeout 60 mcp --name dictcli

7) Servicio systemd en Ubuntu

Unidad: deploy/systemd/dictcli-api.service, script: scripts/run_dictcli_api.sh
