dict-cli Guía completa

CLI de diccionario SQLite offline con modos REST API (serve) y MCP stdio (mcp).

  • Búsqueda offline
  • Modo REST API
  • Integración MCP

D I C T - C L I   M A N U A L

1) Funciones

  • Búsqueda por archivo/directorio SQLite
  • Comandos: exact, prefix, meaning
  • Modos: CLI / REST API / MCP

2) Requisitos

  • Go 1.25+
  • Ruta de DB de diccionario
  • /path/to/dictionary_fts5.db o /path/to/dicts

3) Ejemplos CLI

./bin/dictcli --db /absolute/path/to/dictionary-data/dictionary_fts5.db groups
./bin/dictcli --db /absolute/path/to/dictionary-data/dicts --group ko-en exact --term 사전

4) Configuración

  1. --db
  2. DICTCLI_DB_PATH
  3. config.ini

5) REST API

./bin/dictcli --db /absolute/path/to/dictionary-data/dicts --group ko-en --limit 20 --timeout 60 serve --host 127.0.0.1 --port 8080

6) MCP

./bin/dictcli --db /absolute/path/to/dictionary-data/dicts --group ko-en --limit 20 --timeout 60 mcp --name dictcli --version 0.1.0

7) Servicio systemd en Ubuntu

Unidad: deploy/systemd/dictcli-api.service, script: scripts/run_dictcli_api.sh
