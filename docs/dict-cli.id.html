dict-cli Panduan Lengkap

CLI kamus SQLite offline dengan mode REST API (serve) dan MCP stdio (mcp).

  • Pencarian offline
  • Mode REST API
  • Integrasi MCP

D I C T - C L I   M A N U A L

1) Fitur

  • Query berbasis file/direktori SQLite
  • Perintah: exact, prefix, meaning
  • Mode: CLI / REST API / MCP

2) Kebutuhan

  • Go 1.25+
  • Path DB kamus
  • /path/to/dictionary_fts5.db atau /path/to/dicts

3) Build

./build.sh
task build
CGO_ENABLED=0 go build -trimpath -ldflags "-s -w" -o bin/dictcli ./cmd/dictcli

4) Test

./test.sh

5) Contoh CLI

./bin/dictcli --db /absolute/path/to/dictionary-data/dictionary_fts5.db groups
./bin/dictcli --db /absolute/path/to/dictionary-data/dicts --group ko-en exact --term 사전

6) Konfigurasi

  1. --db
  2. DICTCLI_DB_PATH
  3. config.ini

7) REST API

./bin/dictcli --db /absolute/path/to/dictionary-data/dicts --group ko-en --limit 20 --timeout 60 serve --host 127.0.0.1 --port 8080

8) MCP

./bin/dictcli --db /absolute/path/to/dictionary-data/dicts --group ko-en --limit 20 --timeout 60 mcp --name dictcli --version 0.1.0

9) systemd Ubuntu

Unit: deploy/systemd/dictcli-api.service, Script: scripts/run_dictcli_api.sh
