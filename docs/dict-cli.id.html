dict-cli Panduan Lengkap

CLI kamus SQLite offline dengan mode REST API (rest) dan MCP stdio (mcp).

  • Pencarian offline
  • Mode REST API
  • Integrasi MCP

D I C T - C L I   M A N U A L

1) Fitur

  • Query berbasis file/direktori SQLite
  • Alur pencarian: default --term dan pencarian makna --query
  • Mode: CLI / REST API / MCP

2) Kebutuhan

  • Go 1.25+
  • Path DB kamus
  • /path/to/dictionary_fts5.db atau /path/to/dicts

3) Contoh CLI

dictcli --db /absolute/path/to/dictionary-data/dictionary_fts5.db groups
dictcli --db /absolute/path/to/dictionary-data/dicts --term 사전
dictcli --db /absolute/path/to/dictionary-data/dicts --term 'dict*'
dictcli --db /absolute/path/to/dictionary-data/dicts --query dictionary

Wildcard: * (urutan bebas), ? (1 karakter).

4) Konfigurasi

  1. --db
  2. DICTCLI_DB_PATH
  3. config.ini

5) REST API

dictcli --db /absolute/path/to/dictionary-data/dicts --limit 20 --timeout 60 rest --host 127.0.0.1 --port 8080

6) MCP

dictcli --db /absolute/path/to/dictionary-data/dicts --limit 20 --timeout 60 mcp --name dictcli

7) systemd Ubuntu

Unit: deploy/systemd/dictcli-api.service, Script: scripts/run_dictcli_api.sh
