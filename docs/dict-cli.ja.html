CLI マニュアル

dict-cli 完全ガイド

オフライン SQLite 辞書 CLI です。ローカル実行に加え、REST API サーバー (serve) と MCP stdio サーバー (mcp) モードを提供します。

  • オフライン辞書検索
  • REST API 提供
  • MCP ツール連携

D I C T - C L I   M A N U A L

1) 機能

  • SQLite DB ファイル/ディレクトリを使った辞書検索
  • 検索コマンド: exact, prefix, meaning
  • 実行モード: ローカル CLI / HTTP REST API / MCP(stdio)

2) 必要条件

  • Go 1.25+
  • 辞書 DB パス（別途準備）
  • 統合 DB: /path/to/dictionary_fts5.db
  • または DB ディレクトリ: /path/to/dicts

3) CLI 実行例

./bin/dictcli --db /absolute/path/to/dictionary-data/dictionary_fts5.db groups
./bin/dictcli --db /absolute/path/to/dictionary-data/dicts --group ko-en exact --term 사전
./bin/dictcli --db /absolute/path/to/dictionary-data/dicts --group en-ko prefix --term dict
./bin/dictcli --db /absolute/path/to/dictionary-data/dicts --group ko-en meaning --query dictionary

4) 設定

  1. --db <path>
  2. DICTCLI_DB_PATH
  3. config.ini の [database] path

5) REST API

./bin/dictcli --db /absolute/path/to/dictionary-data/dicts --group ko-en --limit 20 --timeout 60 serve --host 127.0.0.1 --port 8080
  • GET /healthz, /groups, /exact, /prefix, /meaning

6) MCP サーバー

./bin/dictcli --db /absolute/path/to/dictionary-data/dicts --group ko-en --limit 20 --timeout 60 mcp --name dictcli --version 0.1.0

公開ツール: groups, exact, prefix, meaning

7) Ubuntu systemd サービス

サービスユニット: deploy/systemd/dictcli-api.service

起動スクリプト: scripts/run_dictcli_api.sh
