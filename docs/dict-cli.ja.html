CLI マニュアル

dict-cli 完全ガイド

オフライン SQLite 辞書 CLI です。ローカル実行に加え、REST API サーバー (rest) と MCP stdio サーバー (mcp) モードを提供します。

  • オフライン辞書検索
  • REST API 提供
  • MCP ツール連携

D I C T - C L I   M A N U A L

1) 機能

  • SQLite DB ファイル/ディレクトリを使った辞書検索
  • 検索フロー: 既定 --term と意味検索 --query
  • 実行モード: ローカル CLI / HTTP REST API / MCP(stdio)

2) 必要条件

  • Go 1.25+
  • 辞書 DB パス（別途準備）
  • 統合 DB: /path/to/dictionary_fts5.db
  • または DB ディレクトリ: /path/to/dicts

3) CLI 実行例

dictcli --db /absolute/path/to/dictionary-data/dictionary_fts5.db groups
dictcli --db /absolute/path/to/dictionary-data/dicts --term 사전
dictcli --db /absolute/path/to/dictionary-data/dicts --term 'dict*'
dictcli --db /absolute/path/to/dictionary-data/dicts --term 'com*uter'
dictcli --db /absolute/path/to/dictionary-data/dicts --query dictionary

ワイルドカード: * は任意文字列、? は任意1文字。

4) 設定

  1. --db <path>
  2. DICTCLI_DB_PATH
  3. config.ini の [database] path

5) REST API

dictcli --db /absolute/path/to/dictionary-data/dicts --limit 20 --timeout 60 rest --host 127.0.0.1 --port 8080
  • GET /healthz, /groups, /search

6) MCP サーバー

dictcli --db /absolute/path/to/dictionary-data/dicts --limit 20 --timeout 60 mcp --name dictcli

公開ツール: groups, search

7) Ubuntu systemd サービス

サービスユニット: deploy/systemd/dictcli-api.service

起動スクリプト: scripts/run_dictcli_api.sh
