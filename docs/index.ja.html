使い方を選ぶ

dict-cli

開発・自動化向けの CLI ツールです。辞書結果をターミナルで取得し、REST API と MCP で外部ツール連携ができます。

  • コマンドベースの高速検索
  • アプリ連携向け内蔵 REST API
  • AI / ツール連携向け MCP 機能
  • スクリプトやパイプラインに最適

dictionary デスクトップアプリ

デスクトップで辞書を快適に使うためのアプリです。単語・意味検索や辞書間の比較をシンプルな UI で行えます。

  • デスクトップ UI で素早く検索
  • 複数辞書ソースを横断
  • 直感的なローカル辞書体験
