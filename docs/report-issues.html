문제 신고

버그, 오류, 개선 요청을 편하게 알려주세요

동작 중 불편했던 점이나 개선 아이디어가 있다면 아래 안내를 확인한 뒤 Google Form으로 신고 양식을 작성해 주세요.

신고 전 안내

  • 문제가 발생한 상황을 순서대로 간단히 정리해 주세요.
  • 기대한 결과와 실제 결과를 함께 적어 주세요.
  • 항상 발생하는지, 가끔 발생하는지 재현 빈도를 알려 주세요.
  • 사용 환경(운영체제, 브라우저/앱 버전 등)을 적어 주세요.
  • 개인정보, 민감정보는 제외하고 작성해 주세요.
